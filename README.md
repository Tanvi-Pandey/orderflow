# OrderFlow

> A high-performance stock exchange matching engine and market data gateway built with Java, Spring Boot, LMAX Disruptor, WebSockets, and React.

## 📖 Project Overview

OrderFlow is a real-time trading platform that simulates the core components of a modern stock exchange.

The project focuses on:

- High-throughput order processing
- Low-latency matching engine
- Event-driven architecture
- Real-time market data streaming
- Professional trading dashboard
